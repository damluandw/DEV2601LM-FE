2. CSS
    2.1. Các sử dụng CSS
        + style = ""
        + <style></style>
        + <link src="">
    2.2. Thuộc tính CSS và giá trị thuộc tính CSS
        - systax: 
            tên_thuộc_tính_CSS : giá_trị_thuộc_tính_CSS_1 giá_trị_thuộc_tính_CSS_2 giá_trị_thuộc_tính_CSS_3;
            VD: display: none;
                border: 1px solid #000;
        - một số từ khóa:
            + display,width, height,max/min-width,max/min-height
            + margin/padding
            + text-, font-(thêm số từ khóa khác: font-size, text-algin)
            + border, border-radius
            + position, z-index, top, left, right, bottom, tranditional
            + :before, :after, content
            + :hover
            + color, background
            + ainamtion, ...
        + @font-face, @media
    2.3. Cách tương tác CSS với HTML
        + tương tác thông qua tên thẻ HTML
        + tương tác thông qua tên class của thẻ HTML
        + tương tác thông qua tên id của thẻ HTML
        + tương tác thông qua tên thuộc tính của thẻ HTML
    2.4. Tương tác CSS với HTML nâng cao
        + tag01 tag02
        + tag01>tag02
        + tag01 + tag02
        + tag01 ~ tag02
        + tag01, tag02
        + .....
    2.5. Lưu ý - độ ưu tiên trong CSS
        + CSS viết sau/dưới thì sẽ được ưu tiên sử dụng
        + Nếu cùng 1 thẻ: !important > tên_id > tên_class > tên_thuộc_tính_CSS > tên_thẻ
        + CSS càng trỏ chi tiết hoặc càng gần thì càng được ưu tiên sử dụng
